# 🍱 Community Food Sharing Platform

A web app that helps reduce food waste by connecting people who have extra food with those who need it.

---

## 🚀 Features

* 👤 User Login & Signup
* 🍽️ Post Food Listings
* 🙏 Request Food
* 📊 Dashboard for Donors & Recipients
* 🔔 Real-time Updates

---

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Backend:** Node.js + Express
* **Database & Auth:** Supabase

---

## 📁 Project Structure

community-food-share/
├── food-share-frontend/
├── food-share-backend/

---

## ⚙️ Run Locally

### Frontend

```bash
cd food-share-frontend
npm install
npm run dev
```

### Backend

```bash
cd food-share-backend
npm install
node server.js
```
